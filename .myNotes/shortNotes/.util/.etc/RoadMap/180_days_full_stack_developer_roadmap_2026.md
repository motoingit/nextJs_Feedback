## 🗓️ MONTH 1 – Web Fundamentals (Days 1–30)

- **Day 12:** Flexbox
- **Day 13:** Grid
- **Day 14:** Responsive design principles
- **Day 15:** Media queries
- **Day 16:** Animations & transitions
- **Day 17–18:** Clone a simple website (Landing page)

### 🔹 Git & GitHub (Days 19–22)
- **Day 19:** Git basics (init, add, commit)
- **Day 20:** GitHub workflow
- **Day 21:** Branching & merging
- **Day 22:** Push projects, README writing

### 🔹 JavaScript Basics (Days 23–30)
- **Day 23:** JS intro, variables

- **Day 27:** Functions
