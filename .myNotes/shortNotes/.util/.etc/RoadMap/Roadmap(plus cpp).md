# ⚖️ **C++ vs Java: Key Differences in Topics**

### **✅ What is in C++ (but not Java, or very different):**

* **Pointers** (direct memory manipulation)
* **Manual Memory Management** (`new/delete`, malloc/free)
* **Multiple Inheritance** (Java avoids it, uses interfaces instead)
* **Operator Overloading** (`+` for custom classes, etc.)
* **Preprocessor Directives** (`#define`, `#include`)
* **Header Files** (Java has packages instead)
* **STL (Standard Template Library)** vs Java’s Collections

---

### **✅ What is in Java (extra / different):**

* **JVM & Bytecode** (platform-independent runtime)
* **Garbage Collection** (automatic memory cleanup, no delete)
* **Threads as built-in language feature** (C++ does it via `<thread>` library)
* **Packages** instead of headers
* **Interfaces** (instead of multiple inheritance)
* **Checked Exceptions** (C++ only has unchecked)
* **Reflection** (introspecting classes/objects at runtime)
* **Annotations** (`@Override`, `@Deprecated`)
* **Swing/JavaFX** (GUI frameworks built into Java)
* **Java APIs for networking, DB (JDBC), web apps**
