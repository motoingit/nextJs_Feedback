Nice list already 👍 — you’re clearly **past beginner level**.
Let’s **level this up** with **high-impact VS Code shortcuts** that actually save time (not gimmicks). I’ll group them so they stick.

> **OS assumed: Windows / Linux**
> (Mac equivalents are usually `Cmd` instead of `Ctrl`)

---

# 🚀 Editing & Cursor Power (VERY IMPORTANT)

```txt
Ctrl + D        → Select next occurrence (multi-cursor magic)
Ctrl + Shift + L → Select ALL occurrences of current selection
Ctrl + U        → Undo last cursor selection
```

🔥 Use this instead of Ctrl+F + replace.

---

```txt
Home / End            → Line start / end
Ctrl + Home / End     → File start / end
Ctrl + ← / →          → Jump word by word
```

---

# ✂️ Line Manipulation (daily use)

```txt
Alt + ↑ / ↓        → Move line up/down
Shift + Alt + ↑/↓  → Duplicate line up/down
Ctrl + Shift + K   → Delete entire line
```

---

# 🔍 Navigation (fast code reading)

```txt
Ctrl + P           → Go to file (faster than explorer)
Ctrl + Shift + O   → Go to symbol in file (functions, classes)
Ctrl + G           → Go to line number
```

🔥 `Ctrl + P` + `:` → go to line
🔥 `Ctrl + P` + `@` → go to symbol

---

# 🧠 IntelliSense & Refactoring

```txt
Ctrl + Space       → Trigger suggestions manually
F2                 → Rename symbol (project-wide safe rename)
Ctrl + .           → Quick fix / code actions
```

📌 **F2 is safer than Ctrl+F2** for refactors.

---

# 📂 Tabs, Windows & Layout

```txt
Ctrl + \           → Split editor
Ctrl + 1 / 2 / 3   → Focus editor groups
Ctrl + W           → Close current tab
Ctrl + Shift + T   → Reopen closed tab
```

---

# 🧪 Debugging (even for DSA)

```txt
Shift + F11 → Step out
```
